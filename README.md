## Setup

Install dependencies

```
npm install
```

Configure the firebase in the console, copy and paste the
web config object to the file index.js
Example:
````
const firebaseConfig = {
    apiKey: "",
    authDomain: "",
    projectId: "",
    storageBucket: "",
    messagingSenderId: "",
    appId: ""
};
````

Run the project

```
npm start
```
